# プライバシーポリシー

最終更新日: 2026-07-16

many-ai-usage は、ユーザーの個人情報を収集、保存、販売、共有しません。

## 収集する情報

Cookie、認証情報、トークン、アカウント識別子、閲覧履歴、ページ本文、入力内容、利用状況データを外部へ収集しません。

## 外部通信

独自の外部サーバーやクラウド同期は持たず、取得した利用状況データをアップロードしません。利用状況ページの読み取りと解析はユーザーのブラウザ内で完結します。

初回起動時は、プロバイダーのページにもサンプル台帳にもアクセスしません。ユーザーが **Try samples** の確認画面で同意した場合に限り、`raw.githubusercontent.com/ishizakahiroshi/many-ai-cli` の公開 JSON から URL だけのサンプルを取得します。この通信に Cookie、トークン、閲覧履歴、ページ本文、取得済み usage 値などのユーザーデータは含まれません。JSON は schema 検証し、コードとして実行しません。

## ローカルに保存する情報

登録したプロバイダー設定、教えた selector/fingerprint、正規化したスナップショット、診断サマリをブラウザの拡張機能ストレージに保存します。これらは外部へ送信されません。

## 使用する権限

- `storage`: ローカル設定とスナップショットの保存
- `tabs` / `scripting`: 登録ページの検索とローカル読み取り処理
- GitHub raw への固定 host access: 明示同意後に URL のみのサンプルを取得
- optional host access: ユーザーが許可した登録済み利用状況ページの読み取り

## 変更

このポリシーを変更する場合は、この文書の更新によって通知します。
