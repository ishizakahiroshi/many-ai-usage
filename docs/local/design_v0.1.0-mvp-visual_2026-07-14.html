3. 対処後の状態遷移

対処 → 通常バーへ「昇格」
  • needs_permission → 許可: 上部の通常バー領域に登録順の位置で戻る(位置は最初から予約されている・場所は動かない)
  • needs_permission → 削除: provider が消える。以降 popup にも options にも出ない
  • tile → 再解析で拾えた: 上部の通常バー領域に登録順の位置で戻る
  • tile → 開いて閉じた: 次回訪問時の user visit capture で自動的に snapshot が上書きされる
位置は「登録順で予約席」的な扱い。昇格しても場所は動かない。

5. plan へ反映が要る点(v3 まとめ)

plan §C6 / F2 差し戻し
  • popup 寸法: 幅 560px × 高さ最大 600px(Chrome/Firefox の制限内)。§C6 に明記
