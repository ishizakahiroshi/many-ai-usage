# provider-sample-icons

**Sample-only** letter badges for the starter pack (sister look of many-ai-cli `providerIconHtml`).

- Not official vendor logos or trademarks.
- Not bundled into the extension binary.
- Fetched once when the user imports the starter pack, then stored as local `iconDataUrl`.
- Users may replace each icon in Settings with their own image.
